<h3>HOW WP GUIDANT WORKS</h3>
<p>The plugin is very easy to use in WordPress. Just follow these quick steps-</p>
<ol>
	<li>Install WP GUIDANT and activate it</li>
	<li>Click on the <strong>WP Guidant</strong> on your WordPress dashboard</li>
	<li>You will see a blank page with a button named <strong>‚ÄúCreate New Guide‚Äù</strong></li>
	<li>Click on it and provide name, title, and description for the guide</li>
	<li>Then click on the <strong>Create</strong> button</li>
	<li>Then click on the <strong>‚ÄúCreate New Filter‚Äù</strong> button</li>
	<li>Provide name, title, description, and type of the filter</li>
	<li>You can go with Slider, Card, or Form</li>
	<li>If select <strong>Card</strong>, then set the card type between <strong>radio</strong> and <strong>checkbox</strong></li>
	<li>Then click the <strong>Create</strong> button</li>
	<li>Then click the <strong>‚ÄúCreate New Card‚Äù</strong> button</li>
	<li>Add a label, image, and condition logic for the card filter from <strong>Card Settings</strong></li>
	<li>Then set conditions from the <strong>Card Condition</strong> tab</li>
	<li>Provide condition attribute, matching, value selection, and value</li>
	<li>Hit the <strong>Save Changes</strong> button</li>
</ol>

Appsero SDK **does not gather any data by default.** The SDK only starts gathering basic telemetry data **when a user allows it via the admin notice**. We collect the data to ensure a great user experience for all our users. 

Integrating Appsero SDK **DOES NOT IMMEDIATELY** start gathering data, **without confirmation from users in any case.**

== Installation ==

**The easy way.**

Go to your WordPress Dashboard. Navigate to Plugins > Add New and then search for "WP Guidant". Click on Install and then Activate the Plugin.

That's it, WP Guidant Plugin is now activated on your site!

**The hard way..**

Download "WP Guidant WordPress Plugin"  and then extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
